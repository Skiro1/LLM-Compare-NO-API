# LLM Compare

Простой инструмент для сравнения ответов из разных AI-чатов и ручной сборки итогового синтеза без API и без локальных моделей.

Проект работает в ручном режиме:
- вы пишете исходный запрос,
- улучшаете его через отдельный AI-чат,
- отправляете улучшенный промт в несколько веб-чатов,
- вставляете ответы обратно,
- собираете финальный синтез в одном месте.

## Что умеет

- 5-шаговый workflow: запрос → улучшение → ответы моделей → синтез → результат
- ручное сравнение ответов нескольких AI-чатов в одном интерфейсе
- кнопки `Скопировать и открыть...` для быстрых переходов в нужные чаты
- поддержка индивидуальных системных промтов для отдельных моделей
- простая оценка ответов звёздами и учёт этой оценки в синтезе
- вкладка `Сравнение` для side-by-side просмотра ответов
- вкладка `JSON` для структурированного экспорта результата
- история сессий и продолжение уже начатых чатов
- локальный автосейв в браузере
- импорт и экспорт истории
- редактирование списка моделей в настройках
- настройка системных промтов для улучшения и агрегации
- готовые шаблоны запросов в `Настройки → Шаблоны`
- переключение темы и отключение уведомлений

## Как запустить

### Вариант 1. Просто открыть файл

Откройте [LLM-Compare.html](LLM-Compare.html) в браузере.

### Вариант 2. Через локальный статический сервер

Так обычно надёжнее работают буфер обмена, вкладки и браузерные ограничения.

```powershell
cd d:\aniMeXa
py -m http.server 8000
```

После этого откройте:

```text
http://localhost:8000/LLM-Compare.html
```

## Как пользоваться

1. На шаге `Запрос` введите исходную задачу.
2. На шаге `Улучшение` скопируйте системный промт и промт для улучшения, отправьте их в удобный AI-чат и вставьте улучшенный вариант обратно.
3. На шаге `Модели` отправьте улучшенный промт в разные AI-чаты, затем вставьте ответы в карточки моделей.
4. При необходимости поставьте звёздную оценку ответам.
5. На шаге `Синтез` скопируйте агрегирующий промт, отправьте его в AI-чат и вставьте итоговый ответ обратно.
6. На шаге `Результат` проверьте финальный текст, сравнение ответов и JSON.

## Настройки

В `Настройки` доступны:
- общие параметры
- системные уведомления
- учёт пользовательских оценок в синтезе
- редактирование списка моделей
- настройка системных промтов
- шаблоны для быстрого старта

## Ограничения

- Проект не использует API и не запускает локальные модели.
- Все ответы вставляются вручную.
- Данные хранятся локально в браузере.
- Для загрузки внешних библиотек интерфейса и для переходов в AI-чаты нужен интернет.
- Некоторые браузеры могут ограничивать pop-up окна, если они заблокированы вручную.
